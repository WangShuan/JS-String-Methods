# JS-String-Methods

  1. array 補充: 
      1. splice: 數組的添加/刪除/替換，array.splice(起始下標,修改個數,替換內容)。
      2. join: 把數組裡的元素用join帶入的參數分割 組成字符串。
      3. sort: 把數組裡的元素以字符串的形式比較大小。
  
  2. String 用法:
      1. charAt: 返回指定下標的字符。
      2. indexOf: 返回指定字符首次出現的位置 如果沒有則返回-1。
      3. substring: 擷取某一小段字符串。
      4. split: 把字符串分割成數組。
